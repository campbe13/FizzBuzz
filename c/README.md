# c

Compiled language.

## Install
If you are running a Linux or Unix distro there will be a c compiler installed, usually [gcc](https://gcc.gnu.org/)
## Compile / interpret
```bash
# source in fn.c
# compile
gcc fn.c -o fn
# run
./fn
```
See [this file](RUNTIME.md) for runtime output
